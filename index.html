1. CONFIGURATION



Ce plugin permet à SARAH de trouver et d'annoncer les objects connectés sur votre réseau local. Toutefois, ces objets devront êtres compatibles upnp.

On peut rechercher les objets suivants plusieurs critéres, notament :
  • Suivant le nom de l'objet :

    • Paramètre = friendlyName

  • Suivant le constructeur de l'objet :

    • Paramètre = manufacturer

  • Suivant le modèle :

    • Paramètre = modelName

  • Suivant la description :

    • Paramètre = modelDescription
(D'autres paramètres peuvent êtres utilisés... Mais attention à la vocalisation par SARAH...).

Sachant que certains de ces paramètres peuvent ne pas êtres présents, je vous conseille de rester sur friendlyname, qui est assez "vocalisable".

Ce paramètre doit être stockée dans le fichier FindRemote.prop situé dans le répertoire du plugin.

Pour cela, 2 manières de faire :
  • Renseigner directement l'élément "search" dans le fichier (déconseillé...).
  • Via l'interface web de SARAH en cliquant sur la "tuile" FindRemote (préférable...).
Sachant que si vous n'aurez pas à vous soucier de la "casse" (qui varie suivant les objets), il vous faudra impérativement respecter l'orthographe...

2. DESCRIPTEUR DES COMMANDES



La configuration des requètes vocales passe par le fichier externe : FindRemote.xml, ce fichier est aisément modifiable par vos soins pour en changer le vocabulaire des phrases "clés" utilisées par SARAH.

D'une façon générale, ce fichier permet de déclarer les actions à demander sans avoir à manipuler le code javascript.

Exemples de commandes vocales :
  • SARAH, liste moi les objets connectés.

  • SARAH, scanne le réseau.

  • SARAH quels sont les objets ?
Dans tous les cas, ces commandes n'entrainent qu'une seule et même action de SARAH... Elle vous donnera le nombre, la liste et les 2 derniers digits de l'adresse IP locale de l'objet connecté.
